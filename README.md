## 🧠 Overview

Enterprise-grade document processing pipeline that extracts text (OCR), identifies named entities (NER), extracts tables, auto-summarises, and answers natural language questions about uploaded documents.

---

## ✨ Features

- ✅ Tesseract OCR — PDF, PNG, JPG, DOCX ingestion
- ✅ spaCy NER — persons, organisations, locations, dates, money
- ✅ Table extraction with Pandas output and chart generation
- ✅ RAG-powered document Q&A (LangChain + ChromaDB)
- ✅ Auto-summarisation with key entity highlight
- ✅ Multi-language support (EN, AR, FR, DE, ES)

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/HaniaGhouse0407/Document-Intelligence-Platform.git
cd Document-Intelligence-Platform

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set environment variables (if needed)
cp .env.example .env
# Edit .env with your API keys

# 4. Run
streamlit run app.py
```
